# STM32 Logic Analyzer Learning

## Hardware

- STM32F401RE Nucleo Board
- 24 MHz 8-Channel USB Logic Analyzer
- PulseView
- MAX98357A I2S Audio Amplifier
- ADXL345 Accelerometer

## Objective

This repository documents my learning journey in embedded protocol debugging and verification using STM32F401RE, Logic Analyzer and PulseView.

The goal is to understand low-level communication protocols by implementing them at register level and verifying the generated signals using real hardware.

## Projects Included

### UART_ANALYSIS

Capture and analyze UART communication generated by STM32 using a Logic Analyzer and PulseView.

### I2S_ANALYSIS

Implement I2S communication using register-level programming and verify BCLK, LRC and DIN signals using a Logic Analyzer.

## Skills Learned

- Register-Level Programming
- Logic Analyzer Usage
- PulseView Analysis
- UART Protocol Verification
- I2S Protocol Verification
